Perintah di terminal untuk menjalankan laravel

- composer install
- npm install
- npm run dev
- cp .env.example to .env (samakan database dengan kepunyaan)
- php artisan key:generate
- php artisan migrate
- php artisan storage:link
- php artisan cache:clear
- php artisan serve
